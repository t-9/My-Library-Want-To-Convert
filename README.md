# My-Library-Want-To-Convert
我がライブラが変換せしめん

## Introduction

平文を中二病文に変換します。

## Examples

| 入力      | 出力      |
|:---------|:----------|
| 左手が痛い | 我が弓手が疼く |
| ある晩 | あるNacht |
| そんな約束してないよ | そのような契約を交わした覚えなどないぞ |
